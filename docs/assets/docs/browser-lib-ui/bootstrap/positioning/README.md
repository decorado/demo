## Positioning elements

### Text center
Aligns text in the center

```html
  <div class="text-center">Text center</div>
```

```css
  .text-center {
    text-align: center;
  }
```

### Text right
Aligns text to the right

```html
  <div class="text-right">Text right</div>
```

```css
  .text-right {
    text-align: right;
  }
```
