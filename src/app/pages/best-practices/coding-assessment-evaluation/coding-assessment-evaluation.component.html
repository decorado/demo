CODE ASSIGNMENT

Requisitos funcionais

Desenvolver aplicação web contendo os seguintes requisitos:

- A aplicação deverá estar protegida por login e senha

- Deverá existir 2 tipos de perfis (ADMIN e USER)

- Deverá possuir interface para pesquisa dos usuários cadastrados

- Deverá possuir interface para cadastro dos usuários da aplicação

- O usuário com acesso ADMIN poderá executar cadastro de outros usuários, e visualizar os usuários

- O usuário com acesso USER poderá apenas visualizar os usuários do sistema

- Excluir usuários (A ser incluido nos proximos, pelo Daniel)

Requisitos não funcionais

O frontend deverá ser desenvolvido em Angular 6 utilizando Angular CLI.

Utilizar boas práticas encontradas em https://angular.io/docs/ts/latest/guide/style-guide.html

Para estilos da interface utilize o angular material.

Não é necessário backend, pode “persistir” os dados em memória da aplicação.

Build and run

Usar github para publicar seu código.

Documentar processo de build para que a gente suba a aplicação aqui na Decora rapidamente e sem erros. Não utilize outras tecnologias senão as listadas no teste.

Obs: O cargo é front-end developer, capriche nas telas e usabilidade.
